Как запустить игру локально:

1. Распакуйте архив
2. Установите Node.js с сайта https://nodejs.org/
3. Откройте терминал в папке с игрой
4. Выполните команду: npx http-server
5. Откройте в браузере: http://localhost:8080

Примечание: Игра должна запускаться через веб-сервер, простое открытие index.html в браузере не сработает из-за ограничений безопасности при загрузке модулей.
